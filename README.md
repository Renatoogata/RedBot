Descrição 
-----------
  O produto a ser apresentado será capaz de medir o nícel de ingestão alcoolica que uma  pessoa ingeriu através de seu sensor de gás MQ-3 e irá disponibilizar os dados em uma Display Oled.


Plataforma 
-----------

Arduino Uno
